# Harambe job portal

   


## Installation guidance
 
 Clone [this](https://github.com/nahimanajz/jobportal) repository

```
 cd jobportal-fe
 npm install
 cp .env.example .env
 npm run dev

 ```
 ## Features

 - Create, read update and delete application
 - Read Jobs
 - Signup and signin a user 

## Tools

 - NextJs
 - Tailwincss
 - Typescript
